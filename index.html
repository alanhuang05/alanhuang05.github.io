Robust Deepfake Detection via Synthetic Domain Alignment

Proposes Synthetic Domain Alignment (SDA) to improve deepfake detector robustness against domain shifts — using an unconditional generative model to synthesize samples across source and target domains without requiring direct access to the original source data.

Projects

2026

CS129 / Multi-Label Hate Crime Bias Classification

Trained and compared five classifiers — Logistic Regression, Decision Tree, Random Forest, XGBoost, and a majority baseline — to predict bias motivation category (race, religion, sexual orientation, etc.) across 32,508 California hate crime incidents. Tackled severe class imbalance with SMOTE and stratified splitting. Built for Stanford's ML class (CS129: Applied Machine Learning).
